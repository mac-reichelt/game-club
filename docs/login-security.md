# Login Security

## Password Recovery

As of version [add version here], the login page includes a **Forgot password?** link. This link directs users to a dedicated password recovery page.

- On the login screen, click **Forgot password?** to access the recovery page.
- The recovery page instructs users to contact their club admin to reset their password.
- Users can return to the login screen via the **Back to sign in** link.

### Accessibility

- The **Forgot password?** link uses `aria-label="Reset your password"` for improved accessibility.
